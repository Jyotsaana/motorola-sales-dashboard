# Motorola Sales Dashboard | Power BI

## Project Overview
Interactive Power BI dashboard for analyzing Motorola mobile sales performance across products, cities, customer ratings, and payment methods.

## Key Features
- KPI Cards for Total Sales, Transactions, Customers, and Units Sold
- City-wise Sales Analysis using Map Visuals
- Monthly Sales Trend Analysis
- Customer Ratings Distribution
- Payment Method Analysis
- Mobile Model Performance Comparison
- Interactive Slicers and Filters

## Tools Used
- Power BI
- DAX
- Power Query
- Excel

## Skills Demonstrated
- Data Cleaning
- Data Modeling
- Data Visualization
- Dashboard Development
- Business Intelligence
- KPI Reporting
- DAX Calculations
